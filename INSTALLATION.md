# 📖 Complete Installation Guide for ZEUSosX_044

## 🎯 Overview

This guide will walk you through installing the ZEUSosX_044 theme for Windows 11 File Explorer using Windhawk.

---

## ✅ Prerequisites

- **Windows 11** (build 24H2 or 25H2 recommended)
- **Administrator access** (may be required for Windhawk)
- A **supported DPI scaling** (100%, 125%, 150%, 175%, 200%, or 225%)

---

## 📥 Installation Steps

### Step 1: Download and Install Windhawk

1. Visit https://windhawk.net/
2. Download the latest version
3. Run the installer
4. Complete the installation process
5. Launch Windhawk

**Note:** Windhawk may require administrator privileges. If prompted, allow it.

---

### Step 2: Install the Windows 11 File Explorer Styler Mod

1. In the Windhawk main window, click the **"Explore"** button
2. In the search bar, type: `windows-11 File Explorer styler`
3. Find the mod in the results
4. Click the **"Install"** button

**Status check:** After installation, you should see the mod listed in Windhawk's main interface.

---

### Step 3: Backup Your Current Settings (Optional)

**Recommended:** Before applying the new theme, save your current settings.

1. Click on **"windows-11 File Explorer styler"** in the main Windhawk window
2. Click the **"Settings"** tab
3. Click **"Textual mode"** button
4. Select all text (Ctrl+A)
5. Copy it (Ctrl+C)
6. Paste into a Notepad file and save it
7. Close the text editor

**Why:** This way, you can revert to your previous settings if needed.

---

### Step 4: Access the Mod Settings

1. In the Windhawk main window, locate **"windows-11 File Explorer styler"**
2. Click on it to select it
3. Click the **"Settings"** tab at the top

You should now see the settings panel for the mod.

---

### Step 5: Switch to Textual Mode

1. In the Settings panel, scroll down to find the **"Textual mode"** button
2. Click **"Textual mode"**

A text editor window should open, showing the current YAML configuration (or it may be empty/default).

---

### Step 6: Clear the Current Configuration

1. In the text editor, select all text (Ctrl+A)
2. Delete it (Delete or Backspace)
3. The editor should now be completely empty

---

### Step 7: Copy the ZEUSosX_044 YAML Code

**Option A: From This GitHub Repository**

1. Open this repository in your browser
2. Click on the file **`ZEUSosX_044.yaml`**
3. Click the **"Copy raw contents"** button (or Ctrl+C if selected)

**Option B: From the README**

1. Locate the YAML code section in the README
2. Select and copy it

---

### Step 8: Paste the YAML Code

1. Click in the text editor window
2. Paste the code (Ctrl+V)

The entire YAML configuration should now be visible in the editor.

---

### Step 9: Save and Apply

1. Look for and click the **"Save settings"** button
2. Wait a moment for the changes to process

**Success indicators:**
- ✅ File Explorer windows immediately resize to 44px height
- ✅ The layout appears more compact
- ✅ No error messages appear in Windhawk

---

### Step 10: Verify the Installation

1. Open a new File Explorer window (Windows+E)
2. Observe the changes:
   - Navigation bar should be much more compact
   - Address bar and search box appear in a single row
   - Overall height should be approximately 44px
   - The Mica translucent effect should be visible

**If it doesn't work:**
- See the **Troubleshooting** section below

---

## 🔧 Troubleshooting

### Issue: Changes Don't Take Effect

**Solution:**
1. Make sure you clicked **"Save settings"** ✓
2. Close all File Explorer windows (completely)
3. Wait 5 seconds
4. Open a new File Explorer window

### Issue: Theme Looks Different at My DPI Scale

**Cause:** ZEUSosX_044 has been tested at multiple scales, but Windows rendering may vary.

**Solution:**
1. Verify your DPI setting matches one of the tested scales:
   - 100%, 125%, 150%, 175%, 200%, or 225%
2. Restart File Explorer (close all windows)
3. Check if the issue persists

### Issue: Address Bar is Completely White

**This is a known bug (see Known Bugs section in README)**

**Solution:**
1. Don't open folders directly from Desktop
2. Instead, use:
   - This PC (My Computer)
   - Quick Access / Pinned shortcuts
   - Taskbar File Explorer shortcut

### Issue: I Can't Drag the Window

**This is expected behavior (see Known Bugs section in README)**

**Solution:**
- Click on the **Search Box icon** area to drag
- Or click on the **empty space** between the Close button and Search Box

### Issue: Tabs Are Invisible

**This is expected behavior (see Known Bugs section in README)**

**Solution:**
- Don't use "Open in new tab" from context menus
- Always use **"Open in new window"** instead

### Issue: I Want to Revert

1. Restore your backed-up settings file (from Step 3)
2. Go to Settings → Textual mode
3. Clear everything
4. Paste your old YAML code
5. Click "Save settings"
